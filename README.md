
# 🪧 SignCalc - Akıllı Tabela Fiyat Otomasyonu



## 📌 Proje Hakkında



**SignCalc**, ışıklı tabela, kutu harf ve reklam üretimi sektörü için geliştirilmiş **otomatik fiyatlandırma ve optimizasyon** yazılımıdır.



### 🎯 Temel Özellikler



| Özellik | Açıklama |

| :--- | :--- |

| **Görüntü İşleme** | OpenCV ile kontur analizi, alan ve çevre hesaplama |

| **Çoklu Format** | PNG, JPG, SVG, DXF, PDF, AI, EPS |

| **Ölçek Kalibrasyonu** | 2 nokta tıklama ile piksel → mm dönüşümü |

| **Nesne Ayrıştırma** | Her harf/logoyu tek tek tespit etme |

| **Fiyat Motoru** | Malzeme + İşçilik + Fire + Kar otomatik fiyat |

| **Nesting** | Malzeme israfını minimize etme |

| **Raporlama** | PDF ve Excel teklif çıktısı |

| **Veritabanı** | SQLite ile malzeme fiyatları ve ayarlar | EOF
